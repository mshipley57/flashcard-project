Javascript

an object-oriented computer programming language commonly used to create interactive effects within web browsers

Operators

a symbol or function denoting an operation

Variables

a data item that may take on more than one value during the runtime of a program.

Variable naming conventions

e.g. camelCase: a typographical convention in which an initial capital is used for the first letter of a word forming the second element of a closed compound

Functions

a named section of a program that performs a specific task. In this sense, a function is a type of procedure or routine. Some programming languages make a distinction between a function, which returns a value, and a procedure, which performs some operation but does not return a value.

Methods

Data is represented as properties of the object and behavior as methods

Arguments

a value that is passed between programs, subroutines or functions. Arguments are independent items, or variables, that contain data or codes. When an argument is used to customize a program for a user, it is typically called a "parameter."

Parameters

a parameter (often called formal parameter or formal argument) is a special kind of variable, used in a subroutine to refer to one of the pieces of data provided as input to the subroutine

Return

The return statement terminates the execution of a function and returns control to the calling function. A return statement can also return a value to the calling function.

Chaining methods

also known as named parameter idiom, is a common syntax for invoking multiple method calls in object-oriented programming languages.

Strings

a string is any finite sequence of characters (i.e., letters, numerals, symbols and punctuation marks). An important characteristic of each string is its length, which is the number of characters in it. The length can be any natural number (i.e., zero or any positive integer).

Boolean

a binary variable, having two possible values called “true” and “false.”

Undefined

a condition where an expression does not have a correct value, although it is syntactically correct. An undefined value must not be confused with empty string, boolean "false" or other "empty" (but defined) values.

NaN

standing for not a number, is a numeric data type value representing an undefined or unrepresentable value, especially in floating-point calculations.

Escape

used for any of various functions, as to interrupt or cancel the current process or running program, or to close a pop-up window. Expand.

Alert

The alert() method displays an alert box with a specified message and an OK button.

Comments

To create a single line comment in JavaScript, you place two slashes "//" in front of the code or text you wish to have the JavaScript interpreter ignore. When you place these two slashes, all text to the right of them will be ignored, until the next line.

jQuery

jQuery is a concise and fast JavaScript library that can be used to simplify event handling, HTML document traversing, Ajax interactions and animation for speedy website development. jQuery simplifies the HTML's client-side scripting, thus simplifying Web 2.0 applications development.

Attributes

The attributes property returns a collection of the specified node's attributes, as a NamedNodeMap object.
